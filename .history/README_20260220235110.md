## Python Setup (Windows)

If you want to run the notebooks locally, use a virtual environment and install dependencies:

1. Ensure Python 3.10+ is installed (3.8+ is supported).
2. Create a venv and activate it (PowerShell):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

3. Upgrade pip and install dependencies (we filter out a non-standard `sqlite3-python` entry if present):

```powershell
.\.venv\Scripts\python -m pip install --upgrade pip setuptools wheel
Get-Content requirements.txt | Where-Object {$_ -notmatch 'sqlite3-python'} | Out-File -Encoding utf8 requirements-fixed.txt
.\.venv\Scripts\python -m pip install -r requirements-fixed.txt
```

4. Start JupyterLab and open the notebook `notebooks/[FINAL]_RideWire-Business-Dashboard.ipynb`:

```powershell
.\.venv\Scripts\python -m jupyter lab
```

Notes:
- If installation fails on a specific package, check the package name in `requirements.txt`.
- Use `requirements-fixed.txt` if you need to remove problematic entries.

# Getting Started with RideWire

## Installation

### Prerequisites
- Python 3.8 or higher
- pip (Python package installer)
- Git

### Setup

1. Clone the repository:
```bash
git clone https://github.com/STEPHENIESGEM/ridewire.git
cd ridewire
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Open Jupyter and explore:
```bash
jupyter notebook
```

## Directory Structure

```
ridewire/
├── notebooks/          # Jupyter notebooks
├── docs/              # Documentation
├── src/               # Python source code
├── examples/          # Example scripts
└── assets/            # Images, logos, diagrams
```
